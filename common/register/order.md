# 订单问题

### 不同授权的音乐音质有区别吗？

不同授权的音乐音质**没有区别**。

### 付款成功之后应当如何下载我购买的音乐？

在支付成功后，用户在相应的订单的“订单详情”可进行购买的音乐下载。

### 库音提供的音频文件是什么类型的？

购买成功后，用户可以下载 wav 格式的无损音频文件（不含有音频水印）
