# 注册问题

### 如何进行注册

你可以通过“手机验证码”的方式注册库音账户。

“手机验证码”注册方式需要用户提供正确的手机号以及收到的短信验证码以供注册。

### 我的手机收不到注册确认短信怎么办？

1. 请检查手机号码输入是否正确；
2. 请检查手机自带拦截功能或安全软件是否将验证短信拦截；
3. 请检查手机信号是否正常。
